# Vinted Backend Homework Assignment

## Description
Shipment discount calculation module.

## Prerequisites
- Python 3.12.4

## Usage
1. Run the main script:
    navigate to the project directory and run the following command:
    ```py main.py ```

## Testing
1. Run the tests:
    navigate to the project directory and run the following command:
    ``` py -m unittest discover -s tests ```
